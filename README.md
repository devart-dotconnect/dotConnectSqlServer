### Overview
------------

[dotConnect for SQL Server](https://www.devart.com/dotconnect/sqlserver/) is a versatile enterprise-grade connector that enables access to SQL Server data in .NET applications of all levels. It delivers broad compatibility with SQL Server versions and .NET platforms, and easy integration with a number of IDEs, including Visual Studio.

Moreover, this connector regularly undergoes security and performance tests to make sure it stays ahead of the competition at all times. Extra benefits include priority support, frequent updates, and detailed documentation to help you with a quick and easy start from day one.

* Access to data with no need for any client libraries
* Full compliance with ADO.NET standards
* Secure connections to SQL Server via TCP/IP
* Support for SQL Server-specific data types
* Integration with Visual Studio
